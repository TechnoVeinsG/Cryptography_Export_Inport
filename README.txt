I. 名称
　非対称鍵ジェネレータ および RSA暗号のサンプル

II. 説明
　次の処理を行う方法を示すサンプルプログラムです
　・暗号化する際に必要な公開鍵を生成する方法
　・非対称鍵ペアをエクスポート、インポートする方法

　1. プログラム開発について
　　1) 公開鍵ファイルについて
　　　本プログラムで使用しているライブラリは Visual Studio に標準添付のものを利用しております
　　　そのため、公開鍵ファイルはXML形式になります
　　　一般的なOpenSSL形式などではありませんのでご注意ください

　　2) 秘密鍵の管理について
　　　本プログラムでは秘密鍵の格納先としてキーコンテナを利用しています

　　3) その他
　　　本プログラムは処理の概要を簡潔に示すことを目的としているため、エラー処理など割愛している部分が多くあります
　　　本プログラムを参考にして開発される際は、ご注意ください

　2. 使い方
　　a. 非対称鍵を作成する
　　　1) キーコンテナ名を入力する
　　　2) 「公開鍵を作成する」ボタンを押す
　　　3) 保存先フォルダとファイル名を決める
　　　4) 「保存」ボタンを押す
　　　キーコンテナに非対称鍵ペアが格納され、指定したファイルに公開鍵が保存されます

　　b. キーコンテナから非対称鍵ペアを削除する
　　　1) キーコンテナ名を入力する
　　　2) 「キーコンテナから削除する」ボタンを押す
　　　キーコンテナから非対称鍵ペアが削除されます

　　c. キーコンテナから非対称鍵ペアをエクスポートする
　　　1) キーコンテナ名を入力する
　　　2) 「キーコンテナからエクスポートする」ボタンを押す
　　　3) 保存先フォルダとファイル名を決める
　　　4) 「保存」ボタンを押す
　　　キーコンテナから非対称鍵ペアの情報を指定したファイルに保存されます

　　b. キーコンテナに非対称鍵ペアをインポートする
　　　1) キーコンテナ名を入力する
　　　2) 「キーコンテナにインポートする」ボタンを押す
　　　3) エクスポートされた非対称鍵ペアファイルを選択する
　　　4) 「開く」ボタンを押す
　　　キーコンテナに非対称鍵ペアがインポートされます

　　d. 暗号化
　　　1) キーコンテナ名を入力する
　　　2) 平文に文字列を入力する
　　　3) 「暗号化 ↓」ボタンを押す
　　　暗号文に暗号化した値が表示されます

　　e. 復号
　　　1) キーコンテナ名を入力する
　　　2) 暗号文に暗号化された文字列を入力する
　　　3) 「復号 ↑」ボタンを押す
　　　平文に復号化した値が表示されます
